# Sistema de Cadastro e Controle de Pallets
Controle de Vencimento de Supermercado / Front-End

<p style="text-align: justify;">Este projeto foi meu primeiro esforço de praticar os fundamentos de **HTML**, **CSS** e **JavaScript** na construção de uma aplicação front-end. O objetivo principal era criar uma interface gráfica para cadastrar, localizar e gerenciar pallets (conjuntos de produtos), incluindo informações essenciais como: </p>

- Localização do pallet
- Produtos contidos no pallet
- Data de vencimento de cada produto

<p style="text-align: justify;">A ideia central do sistema era permitir que o time de vendas e o setor responsável pudesse monitorar os vencimentos, recebendo alertas e assim tomando as providências necessárias para evitar perdas ou problemas com os produtos.</p>

<p style="text-align: justify;">Embora o projeto tenha sido iniciado com foco na interface gráfica, ele ainda estava em desenvolvimento e não foi finalizado. Ainda era necessário aprofundar conhecimentos em back-end para implementar funcionalidades completas, como armazenamento de dados e notificações automáticas.</p>

<p style="text-align: justify;">Infelizmente, durante o período de desenvolvimento, minha jornada na empresa foi interrompida, e não tive a oportunidade de concluir e apresentar o trabalho completo. Mesmo assim, esse projeto representou um passo importante na minha formação, ajudando-me a entender melhor os fundamentos do desenvolvimento front-end e a necessidade de uma visão mais profunda do sistema.</p>

---

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Objetivos do Projeto

- Criar uma interface intuitiva para cadastro de pallets e produtos
- Localizar pallets rapidamente
- Monitorar datas de vencimento dos produtos
- Alertar o time comercial para providências preventivas

## Status

- Projeto em fase inicial, não finalizado
- Focado na prática dos fundamentos de front-end

## Considerações Finais

<p style="text-align: justify;">Este projeto é uma base que pretendo retomar futuramente, aprimorando a integração com back-end e adicionando funcionalidades completas de gerenciamento e alertas automáticos.</p>
